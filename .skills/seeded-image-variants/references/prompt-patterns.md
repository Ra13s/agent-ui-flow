# Prompt Patterns

## Conservative Improvement

```text
Edit this screen into a cleaner, more usable version.
Keep the same page job and most of the current layout.
Reduce clutter, improve hierarchy, and make the primary action easier to find.
```

## Wild Hop

```text
Use this screenshot as the baseline problem, not as a layout to preserve.
Invent three substantially different UI directions for the same user task.
Keep the product constraints fixed.
Do not add generic dashboard panels or decorative filler.
```

## Implementation Repair

```text
Use this implemented UI screenshot as the source.
Move it closer to the selected concept while preserving functional feasibility.
Focus on layout geometry, component proportions, spacing, hierarchy, and color discipline.
```
