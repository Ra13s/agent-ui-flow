# Storyboard Review

Status: first pass from the worked example.

## Findings

```text
B1 / entire beat / no-value text / welcome and paragraph delay the camera; nothing the user does
next changes because of them / cut the beat, open in the camera

B1 / "we'll walk you through the three essential tools..." / overexplanation / announces the
curriculum the user is about to experience beat by beat / delete

B1 / stats: "0 photos improved" / belonging / progress module on a tutorial screen, and a zero stat
on first open reads as an insult / stats live on the profile; remove

B2 / "Tap the shutter button below..." / obvious text / shutter is the only affordance on a camera
screen / delete

B2 / button: "I took it" / flow logic / dead confirmation; the shutter already fired / advance on
shutter

B3 / body paragraph / overexplanation / facts about light before action; the reshoot instruction
carries the useful part / cut body

B4 + B5 / "Great job!" twice / repetition + voice / form-letter praise / praise once, if at all,
naming what changed

B5 / "You've learned about light and angle." / flow logic / timing was promised and never taught;
payoff tells instead of shows / show cold shot next to final shot

B5 / two buttons / flow logic / tutorial payoff splits the next step / one CTA
```

## Accepted fixes

- Kill the welcome beat.
- Start with the camera.
- Delete dead confirmation.
- Add the missing timing beat.
- Make payoff a visual before/after.

None yet.
