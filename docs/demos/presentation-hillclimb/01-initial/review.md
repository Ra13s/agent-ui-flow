# Initial Review

Stage: initial visual review.
Consumed: `01-initial/screenshot.png`.

## Findings

- High / focus conflict / the huge headline, plan card, side stats, and CTAs all compete for first
  glance / make the daily recording task the dominant object.
- High / duplicated information / `4-day streak`, focus, coach, and recap are repeated across header,
  chips, plan copy, and side cards / keep each fact once.
- High / wrong timing / `Coach preview` and `Recap unlocks later` explain a future screen before the
  user records anything / remove future feedback from the entry page.
- Medium / CTA conflict / `Start warm-up`, `Start pitch`, and `Locked` all look actionable / use one
  primary CTA.
- Medium / dashboard creep / practice volume stats belong on progress, not the daily start surface /
  keep only a small secondary snapshot if orientation is needed.
- Medium / typography / hero headline uses landing-page scale for a product task page / reduce type
  and make the scene/task title the focus.

## Direction

The next image round should explore how to make one object dominate: the daily recording task.
